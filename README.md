## ✏️ How to Edit

Everything is in `index.html`. Open it and search for these markers:

| What to change | Search for |
|---|---|
| Profile photo | `src="assets/photo.jpg"` |
| Hero text & tags | `id="hero"` |
| About text | `id="about"` |
| Publications | `id="publications"` |
| Contact links / email | `id="contact"` |
| Color theme | `:root {` at top of `<style>` |

To edit directly on GitHub: click `index.html` → pencil icon ✏️ → edit → **Commit changes**. Live in ~30 seconds.

---

## 🎨 Retheme in 10 seconds

In the `<style>` block, find `:root {` and swap two variables:

```css
/* Current: Navy + Teal */
--navy: #0b1526;   --teal: #2dd4bf;

/* Option A — Midnight + Violet */
--navy: #0d0b1e;   --teal: #a78bfa;

/* Option B — Charcoal + Emerald */
--navy: #0d1810;   --teal: #34d399;

/* Option C — Slate + Sky Blue */
--navy: #0b1320;   --teal: #60a5fa;

/* Option D — Dark + Amber */
--navy: #1a1208;   --teal: #fbbf24;
```

Five full presets (with all dependent variables) are documented as comments at the very top of the CSS — just copy-paste the block you want.
